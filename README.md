# 曲径小兔版 - 微信小游戏 v2

🎮 一款竖屏休闲跑酷类微信小游戏，玩家控制可爱的小兔子在森林道路上前进。

## 🐰 游戏简介

- **主角**：可爱的小兔子
- **目标**：让小兔子在无限延伸的道路上前进，躲避障碍
- **操作**：点击左/右按钮控制兔子左右移动
- **特色**：自动前进机制 + 连续前进得分加成

## 🚀 本地运行步骤

### 1. 准备工作

确保已安装 **微信开发者工具**（下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html）

### 2. 打开项目

1. 打开微信开发者工具
2. 点击「导入项目」或「打开项目」
3. 选择 `qujing-rabbit-minigame-v1` 文件夹
4. **重要**：确保选择项目类型为「小游戏」

> ⚠️ **如果工具误识别为小程序**：  
> - 打开 `project.config.json`，确认 `compileType` 值为 `"game"`（而非 `"miniprogram"`）  
> - 确保 `minigame/` 目录下存在 `app.json` 和 `game.json` 配置文件  
> - 如仍报错，删除后重新导入项目

### 3. 运行预览

- 点击「编译」按钮或使用快捷键 `Ctrl+B`
- 即可在模拟器中预览游戏效果

### 4. 真机调试

1. 使用手机扫码预览
2. 确保手机和电脑在同一 WiFi 下

### 5. 浏览器预览（可选）

- 可在浏览器中打开 `preview/preview.html` 查看游戏效果（WebAudio 音效在部分浏览器可能受限）

## 🔧 导入后若仍显示 mp 环境

如果导入项目后仍显示「小程序 (mp)」而非「小游戏 (game)」环境，请按以下步骤排查：

1. **清除缓存**
   - 关闭微信开发者工具
   - 删除项目目录下的 `.vscode`、`.weixin` 等隐藏文件夹（如有）
   - 重新打开开发者工具

2. **重新导入**
   - 点击「导入项目」
   - 选择项目根目录
   - **确保选择项目类型为「小游戏」而非「小程序」**

3. **确认 project.config.json**
   - 确保 `compileType` 字段值为 `"game"`
   - 确保 `minigameRoot` 字段指向正确目录（如 `"minigame/"`）
   - 确保 `miniprogramRoot` 字段存在（可为空字符串 `""`）

4. **检查配置文件**
   - 确认 `minigame/game.json` 存在且格式正确
   - 确认 `minigame/app.json` 存在且包含 `pages` 字段

5. **重启开发者工具**
   - 有时需完全退出开发者工具后重新打开

> 💡 如果问题持续，建议删除整个项目文件夹后重新克隆/解压，再导入。

## 🎮 操作说明

- **← 按钮**：小兔子向左跳动
- **→ 按钮**：小兔子向右跳动
- 小兔子会自动向前移动
- 当前方有路时，兔子会自动前进并得分
- 当前方无路时，兔子会掉落，游戏结束

## 🎵 音效说明

v2 采用 WebAudio API 实现音效，无需外部音频文件：

- **前进音效**：660Hz 正弦波，80ms
- **移动音效**：440Hz 三角波，60ms
- **得分音效**：523Hz + 659Hz 上升音阶
- **游戏结束**：400Hz→300Hz 锯齿波

> 💡 **注意**：部分浏览器有自动播放策略限制，需用户交互后才能播放音效。
