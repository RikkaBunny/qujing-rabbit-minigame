### 🔄 核心改动：企鹅 → 兔子

| 原版（企鹅） | 小兔版 |
|-------------|--------|
| 冰雪风格背景 | 森林/草地风格背景（蓝绿渐变） |
| 白色腹部、黑色背部、橙色脚蹼 | 白色身体、长耳朵、粉色鼻子、红色眼睛 |
| 道路：白色积雪格子+蓝色边框 | 道路：棕色泥土格子+绿色草边 |
| 企鹅(gridX, gridY) | 兔子(gridX, gridY) |
| 企鹅向左/右跳动 | 兔子向左/右跳动 |
| 企鹅自动前进 | 兔子自动前进 |
| 企鹅失败检测 | 兔子失败检测 |
