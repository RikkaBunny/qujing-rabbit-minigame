# 《曲径》小兔版 - 游戏开发文档

## 0. 项目结构说明（重要）

- **主运行版本**：微信**小游戏**，入口在 `minigame/` 目录。
  - 用微信开发者工具打开本项目时，请选择/确认 **项目类型为「小游戏」**；`project.config.json` 中 `compileType: "game"`、`miniprogramRoot: "minigame/"`，实际运行的是 `minigame/game.js` → `minigame/main.js`（Canvas 渲染）。
- **小程序代码**：仓库根目录的 `app.json`、`pages/`、`components/` 为**从小程序转换前的源码**，目前仅作保留参考，不参与小游戏构建。
- **逻辑同步**：小游戏玩法逻辑在 `minigame/page-logic.js`，由 `pages/index/index.js` 转换而来。若修改**玩法、计分、道路生成、状态机**等逻辑，建议两边同步维护（或先改 `pages/index/index.js` 再复制到 `minigame/page-logic.js`），避免小游戏与历史版本行为不一致。

## 1. 游戏概述

《曲径》小兔版是一款竖屏休闲跑酷类微信小游戏。玩家控制一只可爱的小兔子，在无限延伸的森林道路上前进。道路会随机向左或向右弯曲，玩家需要通过左右操作让小兔子始终保持在道路上。游戏节奏紧张刺激，考验玩家的反应速度和预判能力。

## 2. 核心玩法

### 2.1 角色与控制
- **角色**：一只萌趣的小兔子，站在格子道路中央
- **操作方式**：屏幕左下角和右下角分别设有 `<-` 和 `->` 两个虚拟按钮
- **控制逻辑**：
  - 点击 `<-`：小兔子向左跳动一格
  - 点击 `->`：小兔子向右跳动一格

### 2.2 自动前进机制
- **当小兔子前方有可通行的格子道路时，小兔子会以固定速度自动向前跳动**
- **当小兔子前方没有可通行的格子时，小兔子停留在当前格子，等待玩家操作**
- 玩家必须通过左右跳动，让小兔子移动到前方有路的位置，才能继续自动前进

### 2.3 道路生成规则
- 道路由**格子**组成，每个格子有两种状态：有路/无路
- 道路每帧/每tick向下移动，速度随游戏时间逐渐增加
- 新行道路生成时，根据前一行的道路位置，进行**随机向左或向右的弯曲偏移**
- 偏移范围：每行相对上一行偏移 -1、0 或 +1 格（保持道路连贯性）

### 2.4 失败条件
- **主要失败条件**：小兔子随道路下移，触碰到屏幕下方边界时游戏结束
- **检测方式**：当小兔子Y坐标 <= 屏幕底部安全区域时触发

## 3. 游戏系统

### 3.1 得分系统
- **基础得分**：每成功前进一格道路，得 1 分
- **连续前进奖励**：连续自动前进时，得分倍率逐渐提升（1x → 2x → 3x...）
- **操作失误惩罚**：每次左右跳动操作后，连续前进倍率重置为1x

### 3.2 难度递增
| 游戏时长 | 道路下移速度 | 道路弯曲频率 | 道路宽度 |
|----------|--------------|--------------|----------|
| 0-30秒   | 慢速         | 低           | 3格      |
| 30-60秒  | 中速         | 中           | 3格      |
| 60-120秒 | 快速         | 高           | 2-3格    |
| 120秒+   | 极快         | 极高         | 2格      |

### 3.3 道具系统（可选扩展）
- **护盾**：抵挡一次掉落危机
- **减速**：短暂降低道路移动速度
- **扩展**：临时增加道路宽度

## 4. 视觉设计

### 4.1 画面风格
- **整体风格**：清新森林风格，Q版卡通渲染
- **配色方案**：
  - 背景：浅绿到深绿渐变（模拟森林草地）
  - 道路：浅棕色泥土格子，带绿色草边
  - 无路区域：透明或浅灰色

### 4.2 角色设计（小兔子）
- 尺寸：约 1.5 格大小
- 外观：白色/灰白色身体、长耳朵、粉色鼻子、红色眼睛
- 动画：
  - 前进时：原地小跳跃动画
  - 左右移动时：跳跃动画
  - 死亡时：摔倒动画

### 4.3 UI 界面
- **顶部**：当前得分、最高分
- **左下角**：`<-` 按钮（半透明圆形）
- **右下角**：`->` 按钮（半透明圆形）
- **游戏结束弹窗**：显示最终得分、重新开始按钮、分享按钮

### 4.4 特效
- 落叶粒子：背景持续飘落的小叶子
- 得分飘字：得分增加时 +1/+2 文字上浮消失
- 道路边缘：发光效果增强道路辨识度

## 5. 音效设计

### 5.1 背景音乐
- 轻松欢快的电子音乐，节奏随游戏速度加快

### 5.2 音效
- **前进音效**：轻快的"嘀"声
- **左右跳动**：跳跃音效
- **得分提升**：金币收集音效
- **游戏结束**：失败的提示音

## 6. 游戏流程

### 6.1 状态机
```
[开始界面] → [游戏中] → [游戏结束] → [重新开始]
                  ↑_______________↓
```

### 6.2 开始界面
- 游戏标题 "曲径"
- "开始游戏" 按钮
- 最高分显示

### 6.3 游戏中
- 道路持续向下移动
- 小兔子自动前进/等待玩家操作
- 实时显示得分

### 6.4 游戏结束
- 弹窗显示最终得分
- 与最高分对比
- "再来一局" 按钮
- 分享功能

## 7. 微信小游戏适配

### 7.1 技术要求
- **游戏引擎**：建议使用 Cocos Creator 或 LayaAir
- **适配**：竖屏 9:16 比例，适配各种手机屏幕
- **性能**：60fps 流畅运行，支持低配机型

### 7.2 分享功能
- 战绩分享：游戏结束后可分享成绩到微信好友
- 邀请好友：分享链接邀请好友一起游玩

### 7.3 数据存储
- **本地存储**：使用 wx.setStorage 保存最高分
- **用户数据**：可选接入用户授权获取昵称头像
